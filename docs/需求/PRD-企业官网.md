# PRD - 企业官网（AI产品展示）

## 项目概述

开发一个企业官网，更好地展示通过AI工具开发的产品，提升品牌形象，吸引潜在客户。

## 背景

- 企业需要官方网站展示产品
- 需要展示AI开发能力
- 需要响应式设计，支持移动端

## 目标

1. 展示企业产品和服务
2. 突出AI技术优势
3. 提供联系方式

## 功能需求

### 1. 首页

- [ ] 导航栏：首页、产品、关于我们、联系我们
- [ ] Hero 区域：企业标语、简介
- [ ] 产品展示：AI产品卡片网格
- [ ] 页脚：版权信息、联系方式

### 2. 产品展示页

- [ ] 产品列表页
- [ ] 产品详情页
- [ ] 产品特性介绍

### 3. 关于我们

- [ ] 企业简介
- [ ] 团队介绍
- [ ] 愿景

### 4. 联系方式

- [ ] 联系表单
- [ ] 公司地址
- [ ] 邮箱电话

## 非功能需求

- 响应式设计（PC + 移动端）
- SEO友好
- 加载速度优化
- 代码可维护性

## 技术栈建议

- 前端：HTML + CSS + JavaScript（可扩展为React/Vue）
- 后端：可选（静态页面也可）
- 部署：GitHub Pages / Vercel
