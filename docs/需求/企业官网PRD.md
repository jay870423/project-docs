## 1. 项目概述

### 1.1 项目背景

随着AI技术的快速发展，公司通过AI工具开发了多款创新产品。为了更好地向外界展示公司的技术实力和AI产品成果，提升品牌形象，吸引潜在客户和合作伙伴，需要搭建一个现代化的企业官网。

### 1.2 项目目标

- **主要目的**：更好地展示通过AI工具开发的产品，突出技术优势和创新能力
- **次要目的**：提升企业品牌知名度，吸引潜在客户，建立行业影响力
- **长期目标**：成为公司产品展示和业务拓展的核心线上入口

### 1.3 目标用户

| 用户角色 | 用户特征 | 使用场景 |
|----------|----------|----------|
| 潜在客户 | 对AI产品感兴趣的企业或个人 | 了解公司产品，寻求合作 |
| 合作伙伴 | 行业内的企业或投资人 | 了解公司技术实力，探索合作机会 |
| 求职者 | 对AI行业感兴趣的从业者 | 了解公司业务，寻找工作机会 |
| 媒体/行业人士 | 关注AI领域发展的观察者 | 跟踪公司动态，了解行业趋势 |

## 2. 功能需求

### 2.1 整体站点结构

```
企业官网
├── 首页
├── 关于我们
├── 产品展示
├── 案例展示
├── 新闻动态
└── 联系我们
```

### 2.2 首页（Home）

#### 2.2.1 导航栏
- **功能说明**：全站统一导航，方便用户快速跳转
- **需求详情**：
  - 左侧显示公司Logo
  - 右侧显示导航菜单：首页、关于我们、产品展示、案例展示、新闻动态、联系我们
  - 移动端自动折叠为汉堡菜单
  - 滚动页面时导航栏吸顶固定
  - 当前页面高亮显示

#### 2.2.2 首屏（Hero Section）
- **功能说明**：展示企业核心定位和价值主张
- **需求详情**：
  - 大号标题：公司/产品名称
  - 副标题：核心价值主张，突出AI开发能力
  - 背景：动态科技感背景或视差滚动效果
  - 1-2个主要CTA按钮：「了解产品」、「联系我们」
  - 向下滚动提示箭头

#### 2.2.3 核心优势介绍
- **功能说明**：展示企业核心竞争力
- **需求详情**：
  - 3-4个核心优势卡片
  - 每个卡片包含图标、标题、简短描述
  - 突出AI驱动、快速交付、创新定制等特点

#### 2.2.4 精选产品展示
- **功能说明**：展示代表性AI产品
- **需求详情**：
  - 展示3-4个核心产品
  - 每个产品包含缩略图/截图、名称、简短描述
  - 点击可跳转至产品详情页
  - 「查看全部产品」按钮链接到产品展示页面

#### 2.2.5 成功案例展示
- **功能说明**：展示客户成功案例
- **需求详情**：
  - 展示2-3个精选案例
  - 每个案例包含客户Logo、案例名称、核心成果
  - 点击可跳转至案例详情页

#### 2.2.6 数据展示
- **功能说明**：增强可信度
- **需求详情**：
  - 已交付项目数量
  - 服务客户数量
  - 合作伙伴数量
  - 技术专利/软著数量（如有）

#### 2.2.7 合作伙伴/客户展示
- **功能说明**：展示合作客户，提升品牌信任
- **需求详情**：
  - 滚动展示客户Logo
  - 灰色调显示，悬停时恢复原色

#### 2.2.8 底部CTA区域
- **功能说明**：引导用户联系
- **需求详情**：
  - 标题：邀请合作
  - 简短描述
  - 「立即咨询」按钮链接到联系我们页面

#### 2.2.9 页脚
- **功能说明**：全站统一页脚
- **需求详情**：
  - 公司简介简短描述
  - 快速链接分组（产品、案例、关于我们、联系我们）
  - 联系方式信息
  - 社交媒体图标链接
  - 版权信息
  - 备案号（如有）

### 2.3 关于我们（About Us）

#### 2.3.1 公司简介
- **功能说明**：介绍公司基本情况
- **需求详情**：
  - 公司发展历程时间轴
  - 公司愿景、使命、价值观
  - 核心团队介绍（头像、姓名、职位、简介）

#### 2.3.2 公司优势
- **功能说明**：详细介绍公司核心竞争力
- **需求详情**：
  - AI开发能力优势
  - 团队技术背景
  - 项目交付经验
  - 服务优势

#### 2.3.3 发展历程
- **功能说明**：展示公司成长历程
- **需求详情**：
  - 按时间顺序展示重要里程碑
  - 可视化时间轴样式

### 2.4 产品展示（Products）

#### 2.4.1 产品列表页
- **功能说明**：展示所有AI产品
- **需求详情**：
  - 卡片式布局展示所有产品
  - 每个产品卡片包含：产品封面图、产品名称、简短描述、标签（产品类型/技术领域）
  - 支持按产品分类筛选（可选）
  - 点击卡片进入产品详情页

#### 2.4.2 产品详情页
- **功能说明**：详细介绍单个产品
- **需求详情**：
  - 产品名称和简介
  - 产品功能特性列表
  - 产品截图/演示图片轮播
  - 技术架构/核心技术点说明
  - AI能力亮点突出展示
  - 应用场景说明
  - 相关案例推荐
  - CTA按钮：「咨询详情」跳转联系我们

### 2.5 案例展示（Cases）

#### 2.5.1 案例列表页
- **功能说明**：展示所有成功案例
- **需求详情**：
  - 卡片式网格布局
  - 每个案例卡片包含：案例封面图、客户名称、案例名称、简短描述
  - 支持按行业/产品分类筛选
  - 点击进入案例详情页

#### 2.5.2 案例详情页
- **功能说明**：详细介绍单个案例
- **需求详情**：
  - 案例背景和需求描述
  - 解决方案设计
  - 使用的AI技术和工具
  - 实施过程
  - 成果展示（数据化效果）
  - 客户评价
  - 相关产品推荐
  - CTA按钮：「类似需求咨询」跳转联系我们

### 2.6 新闻动态（News/Blog）

#### 2.6.1 新闻列表页
- **功能说明**：展示公司新闻和技术文章
- **需求详情**：
  - 列表式布局（左侧图片，右侧摘要）
  - 每篇文章显示：标题、发布日期、摘要、分类标签
  - 分页加载或无限滚动
  - 支持按分类筛选（公司新闻、技术分享、行业观察等）

#### 2.6.2 文章详情页
- **功能说明**：展示文章完整内容
- **需求详情**：
  - 文章标题、发布日期、作者、分类
  - 正文内容支持富文本（图片、代码块、引用等）
  - 上一篇/下一篇导航
  - 相关文章推荐

### 2.7 联系我们（Contact）

#### 2.7.1 联系信息展示
- **功能说明**：展示公司联系方式
- **需求详情**：
  - 公司地址（带地图嵌入，可选）
  - 联系电话
  - 电子邮箱
  - 工作时间
  - 微信/公众号二维码（可选）

#### 2.7.2 在线留言表单
- **功能说明**：方便用户在线提交咨询
- **需求详情**：
  - 姓名（必填）
  - 公司/单位
  - 联系电话（必填）
  - 电子邮箱
  - 咨询类型选择（产品咨询、合作洽谈、招聘咨询等）
  - 留言内容（必填）
  - 提交按钮
  - 提交成功提示

#### 2.7.3 加入我们
- **功能说明**：展示招聘信息
- **需求详情**：
  - 开放职位列表
  - 每个职位包含：职位名称、职责描述、任职要求
  - 简历投递邮箱或按钮

## 3. 用户流程

### 3.1 新用户访问流程

```
用户通过搜索引擎/推广链接进入官网 → 浏览首页了解公司定位 → 查看产品展示了解AI产品 → 查看成功案例验证能力 → 通过联系我们留言咨询
```

### 3.2 潜在客户合作流程

```
访问官网 → 查看案例 → 查看产品 → 联系我们提交需求 → 后台收到咨询 → 跟进对接
```

### 3.3 求职者访问流程

```
搜索公司官网 → 进入关于我们了解公司 → 查看联系我们/加入我们 → 获取招聘信息 → 投递简历
```

### 3.4 媒体/行业人士流程

```
访问官网 → 查看新闻动态 → 了解公司最新动态 → 获取联系方式进行采访对接
```

## 6. 设计风格建议（现代科技感）

### 6.1 整体风格定位

- **设计风格**：现代简约 + 科技感
- **设计关键词**：简洁、专业、科技、创新、未来感
- **视觉方向**：轻量级渐变、微妙动效、空间层次感、AI蓝色调为主

### 6.2 色彩方案建议

#### 主色调
- **主色**：深蓝色/科技蓝（#165DFF 或 #0F172A）
  - 传递专业、科技、可信赖感
- **辅助色**：青蓝色/电光蓝（#06B6D4）
  - 用于强调、按钮、链接，增加科技活力
- **强调色**：浅紫色/渐变色
  - 用于特殊按钮、高亮元素

#### 中性色
- **背景**：
  - 主背景：白色/极浅灰（#FFFFFF 或 #FAFAFB）
  - 区块背景：浅灰（#F5F5F7）
  - 深色区块：深灰/深蓝渐变（#0F172A 到 #1E293B）
- **文字**：
  - 标题：深灰/近黑（#1D2129）
  - 正文：中灰（#4E5969）
  - 次要文字：浅灰（#86909C）
  - 深色背景上：白色/浅灰

#### 渐变建议
- 蓝到青渐变：`linear-gradient(135deg, #165DFF 0%, #06B6D4 100%)`
- 深色科技渐变：`linear-gradient(135deg, #0F172A 0%, #1E293B 100%)`

### 6.3 字体方案

- **主字体**：Inter 或 思源黑体（免费商用）
- **字体层级**：
  ```
  H1: 48-64px (桌面) / 32-36px (移动端)
  H2: 36-48px  / 28-32px
  H3: 24-30px  / 20-24px
  H4: 20px     / 18px
  正文: 16px
  辅助文字: 14px
  小字: 12px
  ```
- **字重**：
  - 标题：600-700
  - 正文：400
  - 强调：500-600

### 6.4 布局与间距

- **栅格系统**：12列栅格
- **容器宽度**：1200px 最大宽度
- **间距规范**：
  - 模块间距：80px - 120px（桌面）
  - 区块间距：40px - 60px
  - 元素间距：8px / 16px / 24px / 32px 基准体系
- **圆角**：轻度圆角（4px - 8px），卡片使用 8px - 12px

### 6.5 科技感设计元素

#### 背景元素
- 网格背景/点阵背景： subtle 的网格或点阵纹理增加科技感
- 渐变光晕：在区块背景使用柔和渐变光晕
- 几何线条：抽象几何线条装饰，不干扰内容
- 动态粒子背景：首屏可使用轻量级粒子动效（性能允许情况下）

#### 卡片设计
- 微妙阴影：`box-shadow: 0 4px 20px rgba(0, 0, 0, 0.08)`
- 悬浮效果：卡片悬浮时轻微上浮 + 阴影加深
- 边框：可选轻微描边 `1px solid rgba(0, 0, 0, 0.06)`

#### 动画效果
- 滚动入场：元素进入视口时淡入 + 平移动画
- 悬浮变换：按钮、卡片悬浮时有微妙缩放/阴影变化
- 渐变流动：按钮背景可使用缓慢渐变流动效果
- 平滑滚动：全站平滑滚动
- 页面切换：页面切换过渡动画（单页应用情况下）

### 6.6 图片风格建议

- **产品展示图**：使用清晰的产品截图，可加上渐变色边框装饰
- **背景图**：使用抽象科技感背景，低透明度不干扰文字阅读
- **团队照片**：保持简洁专业风格
- **图标**：使用线性图标为主，统一线条粗细（2px）
- **AI元素**：适当使用神经网络、节点连接等抽象图形表达AI技术

### 6.7 移动端设计

- **响应式优先**：确保移动端体验良好
- **触控区域**：按钮和可点击元素最小 48px × 48px
- **导航**：移动端使用汉堡菜单，全屏或抽屉式展开
- **图片自适应**：图片根据屏幕宽度自动裁剪压缩
- **字体缩放**：移动端适当缩小字体，但保证可读性

### 6.8 暗黑模式（可选建议）

- 支持一键切换亮色/暗色模式
- 暗色模式采用深灰蓝背景，保护视力
- 文字对比度保持可访问性标准
